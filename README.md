# Розгортання та запуск застосунку:

1. Завантажити репозиторій **git clone https://github.com/lhorneta/tat.git**

2. Перейти до папки проекту **cd tat/**

3. Встановити node_modules
```bash
# using npm
npm install

# OR using Yarn
yarn
```
4. Запуск програми
```bash
# using npm
npm start

# OR using Yarn
yarn start
```

5. Відкрийте [http://localhost:3000](http://localhost:3000), щоб переглянути його у браузері.
